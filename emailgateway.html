Introduction

MCT Email Gateway is an email server solution that provides a full-featured email server with an integrated webmail client. It supports multiple domains, users, and administrators, and can be used with different email protocols such as SMTP, POP3, IMAP, and manages emails using different mail transfer agents (MTA) such as Postfix and Dovecot. MCT Email Gateway is a robust and secure email server solution that provides a range of features for email management. Its easy-to-use interface and flexibility let users in managing any third party application with emails. MCT Email Gateway is an email server solution that provides a full-featured email server stack including email storage, email transfer, and email delivery components. A brief overview on MCT Email Gateway works: install : it on a server. MCT Email Gateway provides a script that automates the installation process, making it easy for even non-technical users to set up an email server. Configuration: Once MCT Email Gateway is installed, it needs to be configured. The configuration process involves setting up email domains, users, and aliases. MCT Email Gateway provides a web-based interface that makes it easy to configure the email server. Email Storage:MCT Email Gateway uses a combination of Postfix, Dovecot, and MariaDB to store emails. Postfix is responsible for receiving emails, while Dovecot provides a secure and scalable IMAP/POP3 server. MariaDB is used to store email user accounts and other email-related data.

How MCT Email Gateway works

Email Transfer : When an email is sent to a user on an MCT Email Gateway server, it is first received by the Postfix mail transfer agent (MTA). Postfix checks the email address and relays the email to the appropriate email server. Email Delivery : Once the email is received by the user's email server, Dovecot provides the email client with access to the email via IMAP or POP3 protocols. Anti-Spam and Anti-Virus : MCT Email Gateway includes built-in anti-spam and anti-virus features to protect the email server from spam and malware. SpamAssassin is used for spam filtering, while ClamAV is used for anti-virus scanning. Webmail: MCT Email Gateway also includes a web-based email client called Roundcube. Roundcube provides users with a simple and intuitive interface for accessing their email from any device with an internet connection.

Key Feature of EGS

SL/TLS encryption

  • MCT Email Gateway supports SSL/TLS encryption for secure communication between email clients and the mail server. This helps protect sensitive information such as passwords and email content from interception.

Spam and virus filtering

  • MCT Email Gateway includes built-in spam and virus filtering to help protect against spam emails and email-borne viruses. The spam filter uses several techniques, such as IP reputation checks, SPF and DKIM validation, and machine learning algorithms to filter out unwanted email.

Password policies

  • MCT Email Gateway allows you to enforce strong password policies for user accounts, such as minimum password length, complexity requirements, and password expiration. This helps prevent unauthorized access to user accounts..

Firewall and fail2ban

  • MCT Email Gateway includes firewall and fail2ban, which can help protect your mail server from unauthorized access attempts. The firewall blocks traffic from suspicious IP addresses, while fail2ban blocks IP addresses that repeatedly fail login attempts.
